Sobre mi

Soy un apasionado del diseño web con experiencia en la creación de sitios web atractivos y funcionales utilizando HTML y CSS. Tengo habilidades sólidas en la implementación de diseños responsivos y en la optimización de la experiencia del usuario. Mi objetivo es seguir aprendiendo y creciendo en el campo del diseño web.


Tengo la capacidad comprobada para crear sitios web visualmente atractivos y fáciles de usar. Experiencia en diseño responsivo y desarrollo móvil. Tengo buena capacidad para la organización y estructura. Esto implica planificar y organizar el contenido de una página web de manera lógica, utilizando etiquetas HTML apropiadas y dividiendo el contenido en secciones claras y distinguibles. - Diseño la interfaz de usuario (UI). Aquí cree el aspecto visual de la interfaz, incluyendo la elección de colores, tipografía, iconos y otros elementos visuales.
